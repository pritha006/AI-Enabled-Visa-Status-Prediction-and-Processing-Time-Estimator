AI-Enabled Visa Status & Processing Time Prediction

This project uses Machine Learning to predict visa application status and estimate processing time based on historical visa data. It is developed as a B.Tech 3rd Year CSE academic project.

Objectives

Predict visa approval status (Classification)

Estimate visa processing time (Regression)

Assist applicants using AI-driven insights

Tech Stack

Scikit-learn
